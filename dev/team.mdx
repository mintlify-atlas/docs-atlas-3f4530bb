## Roles & Responsibilities

### Scrum Methodology

The team follows Scrum methodology to ensure agile and organized project management:

| Role | Team Member | Key Responsibilities |
|------|-------------|---------------------|
| **Scrum Master** | David Orjuela | Sprint planning, Scrum ceremonies, impediment removal, QA validation |
| **Product Owner** | Juan Pablo Álvarez | User story prioritization, backlog management, deliverable validation |
| **Sprint Planner** | Nicolás Sánchez | Sprint backlog organization, repository management, version control |
| **Documentation Lead** | Lucas Rincón | Technical documentation, frontend development, system support |

### Technical Roles

Each team member brings specialized technical expertise:

- **Quality Assurance Lead**: David Orjuela ensures quality standards and functional validation
- **Backend Development**: Juan Pablo Álvarez leads Python backend and probabilistic engine development
- **Frontend Development**: Nicolás Sánchez and Lucas Rincón develop the PyQt-based user interface
- **Documentation**: Lucas Rincón maintains comprehensive project documentation
